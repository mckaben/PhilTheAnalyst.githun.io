Loan Prediction Status Using Machine learning

The goal of this project is to forecast and predict the loan status of applicants and identify
the best model based on accuracy rates. The suggested machine learning algorithms should be

The following are some possible analysis decision points.
• What level of Accuracy is the forecasted loan status from each model?
• Which variables among the selected ones affect the loan status?
• What is the model with the best performance?
• Can the presence of other variables such as loan repayment affect the outcome of the loan
status?
• Is there any correlation between credit history and loan status?.
